# Poston - ソフトボールチーム ホームページ

ソフトボールチームのための公式ホームページです。

## 機能

### 1. 年間予定 (Annual Schedule)
- チームの年間スケジュールを表形式で表示
- 日付、イベント、場所、時間を確認可能

### 2. 試合成績 (Game Results)
- 最近の試合結果をカード形式で表示
- 勝敗、スコア、対戦相手、場所を確認可能

### 3. お問い合わせ機能 (Contact Form)
- 名前、メールアドレス、件名、メッセージを入力
- フォームバリデーション機能付き
- 送信確認メッセージ表示

## 使い方

1. `index.html` をブラウザで開く
2. ナビゲーションメニューから各セクションにアクセス可能

## ファイル構成

- `index.html` - メインHTMLファイル
- `style.css` - スタイルシート（レスポンシブデザイン対応）
- `script.js` - JavaScript（フォーム処理、スムーススクロール）

## 特徴

- レスポンシブデザイン（PC・タブレット・スマートフォン対応）
- スムーススクロール
- モダンなUIデザイン
- フォームバリデーション
